# MovieApp – Visualizador de Filmes em Cartaz

## Descrição

O MovieApp é uma aplicação frontend desenvolvida em React que consome a API pública do TMDb (The Movie Database) para exibir filmes atualmente em cartaz nos cinemas. O projeto permite que usuários visualizem os filmes mais recentes, filtrem por nome e acessem dados como título, nota e data de lançamento, com uma interface responsiva estilizada com Tailwind CSS.

## Funcionalidades

1. Listagem de Filmes em Cartaz:
   - Exibição de pôsteres, título, avaliação e data de lançamento.
   - Interface responsiva com layout em grid.

2. Busca Dinâmica:
   - Campo de busca com filtro em tempo real por nome do filme.

3. Consumo de API:
   - Requisições à API da TMDb com Axios.
   - Suporte a autenticação via API Key ou Bearer Token.
